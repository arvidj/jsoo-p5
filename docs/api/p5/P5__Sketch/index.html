Module P5__Sketch

type t

The sketch. P5 works in two "modes", either taking from the global window object or by passing in a specific "sketch" instance to be used. The latter is more powerful and allows you to have multiple sketches in the window, the only downside is you have to pass the sketch to every single function call.

Every function in this library has an option argument ?sketch which if left out then the global instance is used, otherwise function calls will use that call.
